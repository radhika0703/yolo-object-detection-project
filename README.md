# 🎯 YOLO Object Detection using OpenCV & YOLOv8

This project uses **YOLOv8**, **OpenCV**, and a webcam to perform real-time object detection.

The model detects objects live from the camera feed and displays them with bounding boxes and labels.

---

## 🚀 Features

- Real-time object detection
- Webcam live detection
- Bounding boxes with labels
- YOLOv8 powered detection
- Fast and accurate predictions

---

## 🛠️ Technologies Used

- Python
- OpenCV
- Ultralytics YOLOv8

```bash
pip install ultralytics opencv-python
```

---

## ▶️ Run the Project

```bash
python 2.py
```

---

## 🎥 How It Works

1. Webcam captures live video
2. YOLOv8 processes each frame
3. Objects are detected in real time
4. Bounding boxes and labels are displayed

---

## 📸 Output

The webcam window displays detected objects with:
- Object name
- Confidence score
- Bounding boxes
